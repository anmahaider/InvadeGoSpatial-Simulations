Simulations-Analysis and Visualisation
================
2025-12-17

Base command to run simulations:

    ./invade --u 0.1 --genome MB:2,2,2,2 --rr 4,4,4,4 --grid-x 100 --grid-y 20 --epi-inherit none/dros/ara --console-format summary --basepop 10,1,10 --rep 1 --steps 1 --trigger-defense 40 --mate-radius 1 --selfing-rate 0 --gen 200 --condinv --closed-y

This command will be altered accordingly to create different scenarios
that explore the spatial dynamics of transposon invasions and their
interplay with host defense mechanisms.
